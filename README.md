# 🎮 LoL 자낳대 실시간 경매 시뮬레이터 사용 설명서

본 프로그램은 리그 오브 레전드(LoL) 자낳대(자본주의가 낳은 대회) 스타일의 팀 구성 및 경매를 실시간으로 시뮬레이팅할 수 있는 웹 애플리케이션입니다.

---

## 1. 🚀 서버 실행 방법
본 프로젝트는 **외부 라이브러리 설치가 필요 없는 순수 파이썬 표준 라이브러리** 기반으로 설계되었습니다.

1. **터미널 실행**: 프로젝트 디렉토리(`c:\Users\ha\Documents\자낳대 경배`)로 이동합니다.
2. **서버 실행 명령어**:
   ```bash
   py server.py
   ```
   또는
   ```bash
   python server.py
   ```
3. **웹 접속 주소**:
   - **로컬 주소**: `http://localhost:8000`
   - **LAN 주소 (모바일/타 기기 연동)**: 터미널 창에 출력된 `http://[서버 IP]:8000` 주소로 접속

---

## 2. 👥 역할별 진입 권한
접속 시 최초 게이트 화면에서 세 가지 역할 중 하나를 선택해 진입합니다.

*   **일반 관전자 / 선수 (Viewer)**
    *   실시간 경매 중계 화면, 로스터 상황, 등록된 선수 데이터베이스를 실시간으로 시청 및 조회할 수 있습니다. (조작 권한 없음, 읽기 전용)
*   **팀 감독 (Captain)**
    *   등록된 팀 목록 중 하나의 팀을 선택해 감독 자격으로 입장합니다.
    *   자신이 영입하지 않은 포지션의 선수가 경매에 나왔을 때, **보유 포인트(가용 자본)** 범위 내에서 실시간 **입찰 버튼**을 누를 수 있습니다.
*   **주최자 / 관리자 (Admin)**
    *   중앙 경매 진행 및 데이터 관리를 담당합니다. (비밀번호: 빈 칸이거나 `admin` 입력)
    *   경매 진행, 룰 설정, AI 작동 여부, 팀 추가/삭제, 선수 등록/전체 삭제 등의 시스템적 권한을 모두 가집니다.

## 3. 📂 탭별 상세 기능

### ① 경매 보드 (Auction Board)
실시간 경매가 생중계되는 메인 화면입니다.
*   **선수 카드**: 포지션, 등급(Tier), 소개가 동적으로 연출됩니다. (선수 추첨 시 4초간 무작위 룰렛 애니메이션 및 효과음 재생)
*   **입찰 정보 & 타이머**: 현재 최고 입찰가, 낙찰 유력 감독 정보 및 입찰 갱신 시 초기화되는 10초 타이머가 동작합니다.
*   **실시간 입찰 피드**: 좌하단 영역에서 입찰 참여, 유찰, 낙찰 소식을 타임스탬프와 함께 피드로 송출합니다.
*   **로스터 스냅샷**: 우측 영역에서 모든 팀의 현재 가용 포인트, 슬롯당 평균 보유액, 포지션별 영입 결과를 실시간 요약 중계합니다.
*   **입찰 조작 (감독 전용)**: 버튼을 눌러 지정된 호가 단위(+5p 등)로 즉시 입찰을 시도합니다.
*   **경매 제어 패널 (관리자 전용)**:
    *   `선수 무작위 추첨 시작`: 경매 대기 중인 선수 풀에서 룰렛 연출로 선수를 지정해 개시 대기 상태로 올립니다.
    *   `경매 시작/재개`: 5초 카운트다운 후 본격적인 실시간 입찰을 시작합니다.
    *   `일시정지 / 강제 낙찰 / 강제 유찰`: 돌발 상황 시 경매를 멈추거나 즉시 종료 처리를 강제 집행할 수 있습니다.
    *   `마지막 상태 되돌리기 (Undo)`: 직전의 입찰, 낙찰, 유찰 상태를 안전하게 1단계 되돌리는 복구 기능입니다.

### ② 팀 로스터 (Roster)
*   모든 팀의 포지션별(탑, 정글, 미드, 원딜, 서포터) 영입 현황과 개별 영입 가격을 종합 표로 일목요연하게 비교 분석합니다.

### ③ 선수 리스트 (Player Database)
전체 선수 DB의 현황을 파악하고 관리하는 곳입니다.
*   **조회 기능 (전체 공통)**: 선수의 경매 상태(경매 대기, 진행 중, 낙찰 완료, 유찰됨) 및 최종 낙찰 팀/낙찰 포인트를 한눈에 열람합니다. (비관리자는 읽기 전용)
*   **선수 전체 제거 (관리자)**: "선수 전체 제거" 버튼을 통해 등록된 선수들을 일괄 삭제하여 완전히 새로운 custom 선수 명단을 처음부터 직접 작성할 때 활용합니다.
*   **순서 셔플 및 프리셋 로드 (관리자)**: 대기 중인 선수들의 순서를 섞거나 프리셋(30명) 라인업으로 즉시 되돌릴 수 있습니다.
*   **대기열 바로 올리기 (관리자)**: 무작위가 아닌 특정 선수를 콤보박스에서 골라 바로 다음 경매 대상으로 등판시킵니다.

### ④ 팀 설정 (AI) (Teams Config - 관리자 전용)
*   **새 팀 추가**: 팀명, 감독명, 초기 포인트를 설정하여 실시간으로 경매에 참가할 감독을 무제한으로 확장 등록할 수 있습니다.
*   **팀 삭제하기**: 활성화되지 않은 팀 카드의 하단 "팀 삭제하기" 버튼을 눌러 제외합니다. (해당 팀에 이미 낙찰된 선수가 있었다면 안전하게 대기 상태로 다시 방출 처리됩니다.)
*   **AI 자동 입찰 시스템**: 사람이 부재 중일 때 컴퓨터 AI가 입찰하도록 켜고 끌 수 있습니다. 성향(균형형, 공격형, 가성비 지향)과 남은 보유 잔고 등을 실시간으로 계산해 스스로 긴박하게 눈치 입찰 경쟁에 동참합니다.

### ⑤ 경매 규칙 (Settings - 관리자 전용)
*   **경매 메커니즘 튜닝**: 기본 자본금(포인트), 최소 시작가, 입찰 증가 단위, 타이머 기간 등을 튜닝합니다.
*   **슬롯 포인트 보존 법칙 (Strict Reserve)**: 활성화 시, 남은 빈 포지션들을 최소 가격으로라도 모두 채워 넣을 수 있도록 최소 보존 포인트를 자동 홀딩한 후 입찰 한계를 제한합니다.
*   **효과음 & 한국어 음성 중계(TTS)**: 째깍 소리, 버저음, 팡파레 효과음 및 "OOO 감독, XX포인트 입찰!"과 같은 실시간 기계 음성 아나운서를 켜고 끌 수 있습니다.
*   **시뮬레이터 전체 초기화 실행**: 클릭 시 현재 진행 중인 상태와 로스터, 포인트를 처음으로 안전하게 리셋합니다. (팀 명단 및 선수 목록은 유지됩니다.)

## 4. 💡 유용한 활용 팁
*   **팀 규모 조절**: 4개 팀이나 8개 팀 등 원하는 팀 수만큼 추가/삭제해 맞춤형 대회를 연출할 수 있습니다. 팀 추가/삭제 후 게이트 화면으로 나가면 새로 연동된 감독으로 즉시 선택 진입 가능합니다.
*   **커스텀 리그 빌드**: "선수 전체 제거" 버튼 클릭 후, 나만의 친구들이나 타 스트리머의 선수 정보를 수동 등록하여 즉석 사설 경매 시뮬레이션을 생성할 수 있습니다.
